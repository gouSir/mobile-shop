# 手机商城项目启动指南（小白版）

---

## 一、你需要准备什么

在开始之前，请确保你的电脑上已经安装了以下软件：

| 软件 | 作用 | 如何检查是否已安装 |
|------|------|-------------------|
| **IntelliJ IDEA** | 写代码和运行项目的工具 | 开始菜单搜索 "IntelliJ" |
| **JDK 1.8 或以上** | Java运行环境 | 打开 cmd 输入 `java -version` |
| **MySQL 8.0** | 数据库 | 打开 cmd 输入 `mysql -u root -p` |
| **Maven**（可选） | 项目构建 | IDEA 自带，不需要单独安装 |

> 如果你还没有安装 MySQL，请先去 https://dev.mysql.com/downloads/ 下载 MySQL 8.0 安装包，安装时记住你设置的 **root 密码**。

---

## 二、导入项目到 IntelliJ IDEA

1. 打开 IntelliJ IDEA
2. 点击菜单栏 **File → Open**
3. 选择项目文件夹：`mobile-shop`
4. 点击 **OK**
5. IDEA 会自动识别这是一个 Maven 项目，右下角会弹出提示：
   > "Maven projects need to be imported"
   
   点击 **Enable Auto-Import**（自动导入）

6. 等待 IDEA 右下角的进度条跑完（下载依赖包，可能需要 1-3 分钟）

---

## 三、初始化数据库

### 步骤 1：启动 MySQL 服务

按 `Win + R`，输入 `services.msc`，回车。

在服务列表中找到 **MySQL80**，右键 → **启动**（如果已经是"正在运行"状态就不用管）。

### 步骤 2：执行建库脚本

打开项目文件夹中的 `database/init.sql` 文件。

在 IntelliJ IDEA 中右键点击这个文件 → **Run 'init.sql'**，或者在 MySQL 命令行中执行：

```sql
-- 方法1：用 IDEA 的数据库工具（推荐）
-- 右侧边栏点击 Database → + → Data Source → MySQL
-- 填写：
--   Host: localhost
--   Port: 3306
--   User: root
--   Password: root（你安装MySQL时设置的密码）
--   Database: (先不填)
-- 点击 Test Connection，成功后点 OK
-- 然后打开 init.sql，点击左上角绿色三角执行

-- 方法2：用命令行
-- 打开 cmd，输入：
mysql -u root -p
-- 输入密码后，执行：
source C:\Users\27125\Desktop\ljc的Javaweb作业\mobile-shop\database\init.sql
```

### 步骤 3：验证数据库

执行以下 SQL 确认数据正确：

```sql
USE mobile;
SELECT COUNT(*) FROM products;   -- 应该显示 15
SELECT COUNT(*) FROM users;      -- 应该显示 2
```

> 如果显示 15 个商品和 2 个用户，说明数据库初始化成功！

---

## 四、配置数据库连接（如果需要）

打开 `src/main/java/com/shop/util/DBUtil.java`，检查第 10-12 行：

```java
private static final String URL = "jdbc:mysql://localhost:3306/mobile?...";
private static final String USER = "root";
private static final String PASSWORD = "root";
```

> ⚠️ 如果你的 MySQL root 密码不是 `root`，请把 `PASSWORD` 改成你自己的密码！

---

## 五、启动项目

### 方法 1：用 Maven 插件启动（推荐）

1. 在 IDEA 右侧边栏找到 **Maven** 面板（图标是个 "m"）
2. 展开 **mobile-shop → Plugins → tomcat7 → tomcat7:run**
3. 双击 **tomcat7:run**
4. 等待控制台输出：
   ```
   INFO: Starting ProtocolHandler ["http-bio-8088"]
   INFO: Starting service [Tomcat]
   ```
   看到这行就说明启动成功了！

### 方法 2：用 IDEA 的 Tomcat 配置启动

1. 点击 IDEA 右上角的 **Add Configuration...**（或下拉框 → Edit Configurations）
2. 点击 **+** → 选择 **Tomcat Server → Local**
3. 点击 **Configure** 选择你安装的 Tomcat 目录
4. 在 **Deployment** 标签中点击 **+** → **Artifact** → 选择 **mobile-shop:war**
5. **Application context** 设为 `/mobile-shop`
6. 把 **HTTP port** 改为 `8088`
7. 点击 **OK**，然后点右上角绿色三角 ▶ 启动

## 七、常见问题排查

### 问题 1：启动时报错 "Address already in use"
> 意思是端口 8088 被占用了。

**解决**：打开 `pom.xml`，找到 `<port>8088</port>`，改成其他端口如 `8089`。

---

### 问题 2：访问页面显示 500 错误
> 通常是数据库没连上。

**检查步骤**：
1. MySQL 服务是否启动？→ `Win+R` → `services.msc` → 找 MySQL80 → 启动
2. 密码是否正确？→ 检查 DBUtil.java 中的 PASSWORD
3. 数据库 mobile 是否存在？→ 用命令行连接 MySQL 确认

---

### 问题 3：登录提示"用户名或密码错误"
> 检查数据库 users 表中是否有账号数据。

**解决**：重新执行 `database/init.sql`。

---

### 问题 4：IDEA 中 Maven 依赖报红
> 依赖没有下载下来。

**解决**：
1. 检查网络连接（需要访问 Maven Central 仓库）
2. 点击 IDEA 右侧 Maven 面板的刷新按钮 🔄（Reload All Maven Projects）
3. 如果还不行，检查 pom.xml 中是否有红色波浪线

---

### 问题 5：中文乱码
> 项目已配置 UTF-8 编码过滤器，正常情况下不会乱码。

**如果还有乱码**：
1. IDEA → File → Settings → Editor → File Encodings → 全部设为 UTF-8
2. MySQL 确认使用 utf8mb4 编码
3. Tomcat 的 `URIEncoding` 设为 UTF-8（pom.xml 中已配置）
